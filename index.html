Shar est un reseau social, où vous pourrez partager votre passion, vos créations et rejoindre vos amis en vous inscrivant gratuitement !
LowLib est une bibliotheque de models 3D gratuite créée par mes soins a l'aide du logiciel Blender.
SpyAdv est mon premier jeu avec Unity3D. C'est un FPS où vous devrez survivre a une invasion de zombies.
ProjetFini est un jeu de platforme en 3D. C'est mon premier jeu réalisé avec unity
La Tchatosphere est un site totalement libre d'acces où vous pourrer y discuter avec qui s'y trouvera ! Elle n'est pas accésible pour le moment mais vous pouvez voir le code source ici
Nama est une histoire interactive ou vous serez confrontés a des choix qui déterminerons votre aventure.
Clicker 2 est un jeu tres adictif où le principe est de cliquer sur un minerai pour gagner de nouveaux minerais, puis acheter des améliorations pour avoir de plus en plus de minerais etc.
Délirious est mon premier site ou j'utilisais vraiment le JavaScript (plus préscisément l'API DOM). C'est aussi pour ce site que j'ai découvert Bootstrap. En gros, c'est une histoire ou vous pouvez modifier des données. C'est assez simple mais c'est quand même amusant.
Html Time est mon dernier site avant ce portfolio. C'est un site où je publie des astuces sur des difficultés (plus ou moins grandes) auxquelles j'ai déja été confrontées. J'y explique comment j'ai fait Nama par exemple, mais j'y traite plein d'autres sujets.
